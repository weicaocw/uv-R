# uvr 课程地图（CURRICULUM）

> 方法：自顶向下**理解**全貌，自底向上**建造**。每完成一步就更新本表的进度标记。
> 教学课在 `docs/lessons/`，每步一篇；提交 / PR 中英双语。

## 整体目标
用 Rust 造一个类似 uv / Cargo 的 R 语言包管理器 `uvr`：单二进制、管理 R 版本与项目环境、
lockfile、快速安装。沿途系统学习 **Rust、R 包管理、软件设计**。

## 模块总览（约 32–42 小步，分 7 章）
| 章 | 模块 | 内容 |
|---|---|---|
| **A** | **版本模型** ← 当前 | 解析、比较、约束 |
| B | 元数据 | 解析 PACKAGES / 依赖字段、依赖图 |
| C | 联网 + 缓存 | 外部库、HTTP 抓取、并行、缓存 |

| G | Benchmark + 报告 | 对 pak 的对照实验与报告 |

为"版本号"建模，并赋予它解析、比较、约束的能力——这是后续"依赖求解"的基石。
