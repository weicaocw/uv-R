# uvr 课程地图（CURRICULUM）

> 方法：自顶向下**理解**全貌，自底向上**建造**。每完成一步就更新本表的进度标记。
> 教学课在 `docs/lessons/`，每步一篇；提交 / PR 中英双语。

## 整体目标
用 Rust 造一个类似 uv / Cargo 的 R 语言包管理器 `uvr`：单二进制、管理 R 版本与项目环境、
lockfile、快速安装。沿途系统学习 **Rust、R 包管理、软件设计**。
